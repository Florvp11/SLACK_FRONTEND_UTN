Frontend para una aplicacion de mensajeria basada en slack que permite hacer:

Registrarse,
Login,
Logout,
Crear y borrar Workspaces ,
Crear y buscar channels ,
Enviar Mensajes


Librerias usadas:

React,
React Router DOM,
Vite,
React-icons

**Demo deployed : https://frontend-utn-seven.vercel.app/**
